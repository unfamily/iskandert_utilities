# Angel Block

<ItemImage id="iska_utils:angel_block" />

## What it does

- A **full-sized solid block** that needs **no support**: it stays in mid-air like a temporary scaffold.
- **Right-click in the air** (use the item without targeting another block) to place it **about two blocks ahead** along your view — ideal for bridging gaps or standing on while you build upward.
- You can still **place it normally** against surfaces like any other block item.
