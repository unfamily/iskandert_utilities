# Druidic Agglomeration

<ItemImage id="iska_utils:druidic_agglomeration" />

## Use

Right-click **connected ground** to convert a linked patch into **Druidic Podzol** (circular area, radius 7). Consumes one agglomeration per use.

## Effect

- Ignores natural spread rules: converts connected ground including **entropic** and **druidic** mod blocks already in the patch.
- **Graveyard Soil** is not part of the patch (blocks conversion through it).
- For light spawns, darkness behavior, and redstone acceleration, see **Druidic Podzol**.
