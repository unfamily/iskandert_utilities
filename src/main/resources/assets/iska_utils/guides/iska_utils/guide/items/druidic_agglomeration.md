# Druidic Agglomeration

<ItemImage id="iska_utils:druidic_agglomeration" />

## What it does

Right-click **grass**, **dirt**, or **podzol** to convert a connected patch into **Druidic Podzol** (circular area, radius 7). Only blocks linked through valid targets on the same level are affected. Consumes one agglomeration per use.

See **Entropic terrain** → **Druidic Podzol** for light spawns, darkness idle behavior, and redstone acceleration.

## Crafting

Shapeless recipes (either works):

- <ItemImage id="iska_utils:entropic_agglomeration" /> **Entropic Agglomeration** + **Glowstone Dust** + <ItemImage id="iska_utils:green_sludge" /> **Green Sludge**
- **Entropic Agglomeration** + **Glow Berries** + **Green Sludge**
