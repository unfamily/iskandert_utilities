# Rubber Sap

<ItemImage id="iska_utils:sap" />

## Obtaining

- Tap **filled rubber logs** with a **Tree Tap** or **Electric Tree Tap**.
- Automate with a **Rubber Sap Extractor** (see **Rubber and processing**).

## Uses

- Smelted into **Rubber** (furnace or compatible machines).
- Crafted into a **Rubber Sap Block** for storage.
