Adds some useful things to improve tech modpack player gameplay.

The mod adds:

- Vector Plates: push monsters or players by type of plate, is very useful for mobfarms, it is compatible with cursed earth.
- Portable Dislocator: allow to teleport with nature's compass and explorer's compass.
- Rubber Trees, Rubber Boots (negate fall damage).
- Scanner: uselful for find ores and mobs.
- Infernal Igniter: with a redstone impulse lights the fire in front of him.
- Weather Alterer & Time Alterer: Allow you to control the weather and the day-night cycle.
- Swiss Wrench: a  wrench for turning blocks; it is compatible with most mods.
- Structure System: allows the placement of structures defined by the JSON script system.

Many of the implemented utilties are highly configurable.
